# Linux privat nutzen

## Moegliche Wege

- Linux in VirtualBox ausprobieren
- Linux auf einem ungenutzten Laptop installieren
- Ubuntu neben Windows im Dual Boot nutzen
- Windows komplett ersetzen
- Linux auf einer externen SSD nutzen

## Typische Privatnutzung

- Web
- E-Mail
- Office
- Online-Banking
- Fotos verwalten
- PDF lesen und bearbeiten
- Medien abspielen
- einfache Spiele
- Programmieren und Lernen

## Vor dem Umstieg klaeren

- Gibt es zwingende Windows-Programme?
- Werden bestimmte Drucker, Scanner oder Spezialgeraete gebraucht?
- Sind wichtige Daten gesichert?
- Werden Cloud-Dienste verwendet?
- Gibt es berufliche Vorgaben?

## Gute Umstiegsstrategie

1. Linux in VirtualBox testen
2. Live-USB ausprobieren
3. wichtigste Programme pruefen
4. Daten sichern
5. auf Zweitgeraet installieren
6. erst danach produktiv umsteigen
