# Kurze Geschichte von Linux

## Vor Linux

- Unix entstand in den 1970er Jahren.
- Viele Konzepte moderner Betriebssysteme stammen aus der Unix-Welt.
- Das GNU-Projekt wollte ein freies Unix-aehnliches Betriebssystem schaffen.

## Linux entsteht

- 1991 veroeffentlichte Linus Torvalds den Linux-Kernel.
- Zusammen mit GNU-Werkzeugen entstand ein voll nutzbares freies Betriebssystem.
- Deshalb sprechen viele von GNU/Linux.

## Linux heute

Linux laeuft nicht nur auf klassischen PCs:

- Server
- Smartphones ueber Android
- Router
- Smart-TVs
- Cloud-Infrastruktur
- Supercomputer
- Embedded-Geraete
- Steam Deck

## Merksatz

Linux ist kein Nischenprojekt mehr. Viele Menschen nutzen Linux indirekt jeden Tag, oft ohne es zu merken.
