# Windows, macOS und Linux im Vergleich

## Windows

Staerken:

- sehr verbreitet
- gute Hardware- und Softwareunterstuetzung
- wichtig fuer viele berufliche Anwendungen
- grosse Spieleauswahl

Schwaechen:

- starke Bindung an Microsoft-Konto und Microsoft-Dienste
- Telemetrie und Werbung im System
- Lizenzmodell und Updatepolitik
- fuer aeltere Geraete oft schwerer nutzbar

## macOS

Staerken:

- gut integriertes Gesamtpaket aus Hardware und Software
- lange gute Bedienkonsistenz
- beliebt bei Kreativ- und Medienarbeit

Schwaechen:

- an Apple-Hardware gebunden
- geringe Reparier- und Aufruestbarkeit vieler Geraete
- geschlossenes Oekosystem
- hohe Anschaffungskosten

## Linux

Staerken:

- frei verfuegbar
- viele Distributionen und Oberflaechen
- gute Paketverwaltung
- transparentes System
- oft gut fuer aeltere Hardware

Schwaechen:

- manche Spezialsoftware fehlt
- Hardware kann Sonderfaelle erzeugen
- Umstieg erfordert Lernbereitschaft
- nicht jede Windows-Anleitung passt

## Merksatz

Windows und macOS sind fertige Produktwelten. Linux ist eher ein Baukasten aus Kernel, Distribution, Desktop-Oberflaeche und Anwendungen.
