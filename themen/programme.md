# Wichtige Programme unter Linux

## Alltag

- Firefox oder Chromium: Webbrowser
- Thunderbird: E-Mail
- LibreOffice: Text, Tabelle, Praesentation
- Evince oder Document Viewer: PDF
- Dateien: Dateimanager

## Medien

- VLC: Videos und Musik
- Shotwell oder GNOME Fotos: Fotos verwalten
- GIMP: Bildbearbeitung
- Audacity: Audioaufnahme und Bearbeitung

## Kommunikation

- Signal Desktop
- Element
- Zoom oder Teams ueber Browser oder Paket, falls benoetigt

## System

- App Center: Software installieren
- Aktualisierungsverwaltung: Updates
- Einstellungen: Netzwerk, Bildschirm, Benutzer, Datenschutz
- Terminal: Werkzeug fuer gezielte Befehle

## Kursbotschaft

Viele Alltagsaufgaben funktionieren unter Linux ohne Spezialwissen. Entscheidend ist, Programme ueber vertrauenswuerdige Quellen zu installieren.
