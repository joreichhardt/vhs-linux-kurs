# Installation: USB-Stick, Dual Boot und VirtualBox

## Ziel

Die Teilnehmer installieren Ubuntu entweder direkt auf dem Laptop, neben Windows oder in VirtualBox.

## Installationspfade

### 1. Ubuntu ersetzt das vorhandene System

Geeignet fuer:

- ungenutzte Laptops
- Geraete ohne wichtige Daten
- klare Umstiegsentscheidung

Vorteile:

- einfachster echter Installationsweg
- wenig Partitionierungsaufwand
- kein Dual-Boot-Sonderfall

Risiken:

- vorhandene Daten und Systeme werden geloescht
- Rueckweg nur mit Backup oder Neuinstallation

### 2. Dual Boot mit Windows

Geeignet fuer:

- produktive Privatgeraete mit Backup
- Teilnehmer, die Windows weiter nutzen muessen

Voraussetzungen:

- Backup
- genug freier Speicher
- BitLocker-Recovery-Key
- Windows-Schnellstart deaktiviert
- Speicherplatz in Windows vorbereitet

### 3. VirtualBox

Geeignet fuer:

- produktive Laptops ohne Backup
- Firmen- oder Schulgeraete
- unsichere Ausgangslagen
- erste Tests ohne Risiko

Vorteile:

- keine Veraenderung der echten Festplatte
- leicht zu entfernen
- ideal fuer Experimente

Nachteile:

- weniger Leistung als echte Installation
- Hardware wird nur virtuell durchgereicht
- kein vollstaendiger Alltagstest

## USB-Stick

Empfohlen:

- mindestens 16 GB
- darf geloescht werden
- fuer die Erstellung des Installationsmediums reservieren

## Wichtig

Wenn die Ausgangslage unsicher ist, wird VirtualBox genutzt. Eine echte Installation ist nur sinnvoll, wenn Backup und Risiken geklaert sind.
